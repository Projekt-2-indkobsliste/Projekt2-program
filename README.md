# Projekt2-program
## Vores program
Vores program bliver det nye gode program på markedet blandt familier. Vores program er egentlig bare en indkøbsliste online. Programmet vil kunne bygges videre på ved at inddrage brugere, der kan oprette sig, hvor man derefter kan indskrive de relevante madvarer man vil handle. Vores program er simpel og enhver bruger kan benytte sig af det, da vores program er simpelt og godt. 
## Idéen bag det hele
vi tænkte ummiddelbart at vores program skulle være meget simpelt og ligge meget op af demoen som blev lagt ud, derfor valgte vi en indkøbsliste som vi godt kunne se det praktiske i. Når vores mødre for eksempel spørger om vi vil handle ind efter skole, og giver os et stykke papir med i skolen, med varer vi skal handle ind. Det papir bliver altid på en mærkelig måde væk, og derved kommer vores idé ind. For det smarte ved vores program er, at man som alle i familie kan oprette sig under det samme bruger og derved, kan alle skrive og se hvad der skal handles ind.
## Problemer ved programmet
Efter vi prøvede at tilføje en database, så vores indtastede data ville forbliver der selvom man lukkede appen, stødte vi ind i nogle problemer som gjorde at vores program ikke gad køre, men kunne godt 'bygges' med funktionen i NetBeans.
## Fremtiden for en sådan app
I fremtiden kan vi arbejde videre på programmet, når vi kan lave samarbejde med diverse dagligvarebutikker. Vores programs udvikling vil derfor få nogle nye specifikationer. Programmet vil kunne se priser, tilbud og hvilken butik varen er tilgængelig i. Når vi har lavet de specifikationer, kan vi arbejde videre på, at man også kan betale for varerne i programmet og man kan derfor også vælge afhentningstidspunkt. Dette ligner meget bilka to go, dog kommer appen til at virke ud over alle supermarkeder. Her kan man tilføje en feature, hvor man kan sammenligne priserne med andre supermarkeder. 

Indenfor priserne bruger vi datatyperne. (int) og (double), altså heltal og kommatal. Dette er derfor sammenhængende med varerens værdi. En anden datatype der også kan bliver brugt er (boolean). Hvis vareren ikke er i selve butikken vil den så sige falsk, og derved kan man ikke tilføje den til mad/indkøbslisten.
